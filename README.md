# ParaVibe – Personalized Music Recommendation System

## Overview
Fullstack music streaming platform with AI-based recommendation system using audio and metadata embeddings (MERT, SBERT).

## Architecture
- Frontend: Next.js  
- Backend: NestJS  
- AI Service: FastAPI (Python)

- Frontend: https://github.com/LuanDHV/paravibe-fe
- Backend: https://github.com/LuanDHV/paravibe-be
- AI Service: https://github.com/LuanDHV/paravibe-ai
